  • Hjemmeside for bedriften "Flagolett"

Konsept

For musikere kan det ofte være vanskelig og tidkrevende å finne et lydstudio å spille inn musikk. Da fortjenesten på musikk har endret seg i løpet av årene er det også ofte dyrt å få til et godt opptak. Flagolett tilbyr utleie av Lydstudio med studio teknikker til en fast times pris på en god og oversiktelig hjemmeside. Som en tilleggs tjeneste tilbyr også Flagolett et antall session musikere som kan hjelpe til om det skulle være mangel på band medlemmer under opptak.

Flagolett tilbyr 2 produkter:

  • Utleie av lydstudio
  • Utleie av session musikere

Use cases:

  • Bruker kan lese om bedriften.
  • Bruker kan lese om, se spesifikasjoner og se bilder av produkter.
  • Bruker kan ta kontakt via en contact form.
  • Bruker kan lese nyheter og oppdateringer på siden.

    • Teknologier

    • HTML
    • CSS
    • PHP
    • Javascript / JQuery

    Utover HTML og CSS har vi valgt å bruke PHP for å beholde sidestruktur og lage til en contact form. Javascript og Jquery har vi valgt å bruke for å lage til et easter egg og programmere bildegalleri.
